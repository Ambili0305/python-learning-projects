# Logic Builder - FizzBuzz Program

## Description
A Python program that prints numbers from 1 to 50 using FizzBuzz logic and counts the occurrences of Fizz, Buzz, and FizzBuzz.

## Concepts Used
- Loops (for loop)
- Conditional Statements
- Functions
- Modulo Operator
- Counters

## Learning Outcome
Demonstrates logical problem-solving and control flow implementation in Python.
